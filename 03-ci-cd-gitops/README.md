# Project 3 — CI/CD & GitOps

## Goal
Build a production-grade CI/CD pipeline and GitOps-based Kubernetes deployment.
Focus on automation, troubleshooting, and clean Git history.

## Scope
- CI: build → test → push Docker image
- GitOps: Argo CD deployment from Git
- Incidents: CI failures, broken deployments, rollbacks

## Deliverables
- GitHub Actions workflows
- GitOps manifests (Argo CD + Kubernetes)
- Runbooks per incident
